How It Works
Real-time neural processing

1. Capture — Each video frame is grabbed at native resolution

2. Preprocess — Center-cropped to 224×224, converted to RGB

3. Inference — MobileNet-V3 extracts 512-dim embedding in ~5ms

4. Compare — Cosine similarity vs reference image (0 to 1)

💡 Tip: Set a reference image, then move objects in front of the camera to see real-time similarity changes.
